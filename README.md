A python script and .sh file that run a calculator

The python script must be put in the /usr/local/bin folder
> debian terminal
>
> `sudo cp ~/Downloads/picalc.py /usr/local/bin`
> 
> `sudo cp ~/Downloads/picalc.sh /usr/local/bin`

You can now run picalc.sh in the terminal:

> debian terminal
> 
> `picalc.sh`
